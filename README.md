# Ziartech Company International Sales Dashboard

## Project Overview

An interactive Power BI dashboard developed to analyze sales performance, monitor KPIs, and support data-driven decision-making.

## Objectives

- Monitor Total Revenue
- Analyze Sales by Region
- Compare Store Performance
- Evaluate Sales Representatives
- Track Monthly Sales Trends
- Analyze Customer Demographics

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## Dashboard Features

- Revenue KPI Cards
- Regional Analysis
- Store Analysis
- Sales Representative Performance
- Product Category Analysis
- Customer Gender Analysis
- Monthly Sales Trends
- Weekday Analysis
- Interactive Product Filter

## Key Insights

- ₦19M Total Revenue
- Lagos generated the highest sales
- Lekki was the best-performing store
- Friday had the highest revenue
- Laptop category generated the most revenue
